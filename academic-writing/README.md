# Academic Writing

A comprehensive skill for scholarly writing, research methodology, and academic communication workflows.

## What It Does

Provides systematic guidance and templates for the entire academic writing lifecycle:

- Research question development (FINER criteria)
- Literature review strategy and search design
- Citation management and BibTeX workflows
- IMRaD paper structure (Introduction, Methods, Results, Discussion)
- Academic writing style conventions (tense, hedging, transitions)
- Peer review response templates
- Grant proposal structure (NSF/NIH style)
- Journal selection and cover letter writing
- Research ethics checklists

## Usage

```
/academic-writing
```

Trigger scenarios:
- Writing or restructuring a research paper
- Preparing a literature review
- Drafting a grant proposal
- Responding to peer reviewers
- Selecting a target journal
- Checking research ethics compliance

## Key Features

| Module | Coverage |
|--------|----------|
| Research Design | FINER criteria, question types, refinement workflow |
| Literature Search | Database selection, Boolean strategy, inclusion/exclusion |
| Paper Structure | IMRaD, section-by-section templates, tense guide |
| Writing Style | Hedging, transitions, common problems, paragraph control |
| Peer Review | Response letter template, handling criticism strategies |
| Grant Writing | Specific aims, significance, innovation, budget |
| Publishing | Journal evaluation, red flags, cover letter template |
| Ethics | Human subjects, data management, authorship, misconduct |

## Who It's For

Researchers, graduate students, and academics who want structured guidance for scholarly writing across disciplines.
