## Step 4：DEG差异基因分析（三筛）

### 目的
确认候选亚群在实验组间确实存在转录组水平的差异，而不仅仅是比例变化。同时为后续GSEA提供输入。

### 输入
- Step 3锁定的候选亚群（通常2-3个）
- 各实验条件下的细胞表达矩阵

### 操作
1. 对每个候选亚群，做组间差异分析（如：H_3d vs N_3d）
2. 方法选择：
   - 细胞数充足（>100/组）：Pseudobulk DESeq2（推荐，控制假阳性）
   - 细胞数少（<100/组）：Wilcoxon / MAST（Seurat默认）
3. 每个时间点单独做一次比对（共4次：CON/0d/3d/7d）
4. 火山图展示差异基因分布

### 判断标准
| 指标 | 阈值 | 说明 |
|------|------|------|
| 显著DEG数量 | >50个（|log2FC|>0.25, padj<0.05） | 太少说明该亚群组间差异不大 |
| 上下调比例 | 关注是否有方向性偏移 | 全上调/全下调暗示系统性变化 |
| 时间动态 | 不同时间点DEG数量变化 | 如3d爆发→7d沉默，提示时间窗 |

### 淘汰标准
- 所有时间点DEG数量均<30：该亚群组间差异不显著，降级为辅助亚群
- 仅1个时间点有少量DEG：可能是噪声，需结合GSVA结果综合判断

### 输出
- 火山图（每个亚群×每个时间点，标注关键基因）
- DEG统计表（各时间点上调/下调数量）
- 确认的核心候选亚群（通常1-2个进入GSEA）

### 注意事项
- Pseudobulk方法需要每组至少3个生物学重复
- 如果用Wilcoxon，注意细胞数不平衡导致的假阳性
- DEG结果同时作为后续Marker打分D1维度（log2FC）的数据来源

## Step 5：Pseudobulk + GSEA（锁定核心亚群+核心通路）

### 目的
从通路层面确认核心亚群的功能变化方向，锁定核心通路，并获取leading edge基因作为Marker候选池。

### 输入
- Step 4确认的候选亚群
- Pseudobulk表达矩阵（每个样本聚合为1个profile）
- 基因集数据库（HALLMARK / KEGG / REACTOME / GOBP）

### 操作

#### 5.1 Pseudobulk聚合
1. 按 亚群×样本 聚合细胞表达量（sum或mean）
2. 生成类似bulk RNA-seq的表达矩阵
3. 用DESeq2做差异分析，获取ranked gene list

#### 5.2 GSEA分析
1. 对每个时间点比对（H vs N）分别做GSEA
2. 使用fgsea或clusterProfiler的GSEA函数
3. 多数据库联合分析（建议至少用HALLMARK + 1个细分库）

#### 5.3 结果筛选
1. 按|NES|和padj筛选显著通路
2. 统计各时间点显著通路数量
3. 识别时间动态模式（哪些通路在哪个时间点最显著）

### 判断标准

#### 核心亚群确认
| 指标 | 核心亚群标准 | 辅助亚群标准 |
|------|-------------|-------------|
| 显著通路总数 | >100条 | 10-100条 |
| 最大|NES| | >2.0 | 1.5-2.0 |
| 时间动态 | 有明确的时间模式 | 模式不清晰 |
| 生物学意义 | 与课题核心问题直接相关 | 间接相关或验证用 |

#### 核心通路筛选（逐层收窄）
```
全部显著通路（可能数百条）
  ↓ 按|NES|>2.0 + padj<0.01 筛选
Top 20-30条通路
  ↓ 按生物学相关性 + 时间动态故事性筛选
Top 5-6条核心通路（放主图气泡图）
  ↓ 按|NES|最大 + 故事最完整筛选
Top 2-3条（放山峰图详细展示）
```

#### 通路筛选的优先级原则
1. **与课题核心问题直接相关**（如心肌修复课题优先选代谢/炎症/ECM通路）
2. **时间动态有故事性**（如从激活到耗竭，或从沉默到爆发）
3. **跨亚群有呼应**（如OXPHOS在多个亚群都显著，暗示共性机制）
4. **|NES|绝对值大**（效应量大，更有说服力）

### Leading Edge基因提取
- 从每条核心通路的GSEA结果中提取leading edge基因
- 合并所有核心通路的LE基因，去重
- 统计每个基因出现在几条通路中（Freq/overlap）
- Freq高的基因 = 跨通路hub基因 = Marker候选

### 输出
- GSEA气泡展示图（通路×时间点，NES+显著性）
- 核心通路山峰图（单通路×多时间点并排）
- Leading edge基因列表（含Freq统计）
- 核心亚群+核心通路确认表

### 出图参数（已固化）
**气泡图：**
- X轴=时间点比对，Y轴=通路名，气泡大小=|NES|，颜色=-log10(padj)
- 上调：浅橙→深红/砖红；下调：浅蓝→深紫；不显著：灰色小点

**山峰图：**
- 同一通路2-3个时间点并排（选最能讲故事的对比）
- 标注NES值和padj
